# Connect Four Q-learning Game
This project implements a Connect Four game with an AI opponent powered by tabular Q-learning reinforcement learning. The game features a 6x7 grid where the player (red) moves first against an AI (yellow) that has been trained through 30,000 episodes of self-play. The AI uses a hybrid strategy: Q-learning for training and heuristic rules for gameplay, ensuring it can recognize winning moves, block opponent threats, and prefer center positions. The Pygame interface includes hover preview, real-time win/loss/tie statistics, and console output showing AI reasoning. After training, a learning curve is automatically generated showing the AI's progress from negative to positive rewards. All code is pure Python using only numpy, pygame, and matplotlib - no deep learning frameworks. To run: install dependencies (`pip install pygame numpy matplotlib`) and execute `python connect4.py`.
